# End-to-End DevOps Flask App (Docker + Kubernetes + EKS + CI/CD)

This project demonstrates a complete DevOps workflow: building a Flask app, containerizing it with Docker, deploying it on Kubernetes (EKS), and automating everything using GitHub Actions.

## Topics Covered

- Docker (build, push, run)
- Kubernetes (Deployment, Service, scaling)
- Minikube (local testing)
- AWS EKS (production cluster)
- GitHub Actions (CI/CD)
- IAM & authentication with EKS
- LoadBalancer exposure

## Projects

- Flask App Deployment (Docker → Kubernetes → EKS)
- CI/CD Pipeline with GitHub Actions
